Containers vrs Máquina Virtual

Docker (contenedores)

Virtualiza el sistema operativo: todos los contenedores comparten el mismo kernel del sistema operativo anfitrión, pero pueden tener sus propias bibliotecas y aplicaciones.

Máquina Virtual

Virtualiza el hardware para ejecutar un sistema operativo completo: cada máquina virtual es una instancia completa del sistema operativo con su propio hardware virtualizado.

Fuente: researchgate.net — Virtual Machine versus Docker Container

Ejecucíon de los containers

  • docker ps
  • docker stop [container_id or name]
  • docker start [container_id or name]

Docker Volumenes

  • Los volúmenes de host en Docker son una forma de persistencia de datos en la que se mapea un directorio o archivo específico del sistema operativo
  • Como se usa:
    • docker run -v /path/en/host:/path/en/contenedor imagen

Build & run Image

  • Build:
    • docker build -t ecfm_reduccion_datos:latest .
  • Run
    • docker run -p 8889:8888 -v /home/erick/Documentos/ecfm:/usr/src/app/ ecfm_reduccion_datos
