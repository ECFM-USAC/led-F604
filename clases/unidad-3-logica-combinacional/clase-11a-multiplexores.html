Multiplexores

  • Un multiplexor es un circuito combinacional que selecciona información binaria de una de muchas líneas de entrada y la envía a una sola línea de salida.
  • La selección de una línea de entrada dada se controla con un conjunto de líneas de selección.

Multiplexor y funciones booleanas

  • Un multiplexor es básicamente es un decodificador con una compuerta OR incluida en la unidad
  • Los minitérminos individuales se pueden seleccionar con las entradas de datos.
  • Esto ofrece un método para implementar una función booleana de n variables con un multiplexor que tiene n entradas de selección y 2n entradas de datos, una para cada minitérmino.
